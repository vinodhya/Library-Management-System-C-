# Library Management System (C#)

A C# desktop application developed to manage library operations such as book records, member details, and borrowing processes.

## Features
- Add, update, delete books
- Manage library members
- Issue and return books
- Simple and user-friendly interface

## Technologies Used
- C#
- .NET Framework
- Windows Forms / WPF
- SQL Server 

## Purpose
This project was developed as a student project to practice C# desktop application development.
